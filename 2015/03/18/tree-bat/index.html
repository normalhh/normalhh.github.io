目录树文档生成 Windows 和 MAC 下

发表于 2015-03-18   |   分类于 代码   |  

有的时候，我们需要生成打包的文件目录下的所有文件结构以便于展示给别人。

将下面的代码复制到txt文档中，把后缀文件名改成bat就可以双击运行了。

Windows下：

@echo off
color 2E
MODE con: COLS=40 LINES=20
title %date% %time:~,8%
cls
::================================================================
echo.
echo.
echo 　　BAT当前目录生成很漂亮的结构树-苏
echo.
echo.
echo.
echo 　　选 0   拖动任意目录的结构树
echo.
echo 　　选 1   当前目录的结构树
echo.
echo 　　选 2   每个下级目录文件
echo.
echo 　　选 3   当前目录的目录树
echo.
echo.
set /p p=　请选择(0.1.2.3):　
if %p%==0 goto XX
if %p%==1 goto AA
if %p%==2 goto BB
if %p%==3 goto CC

:AA
CLS
rem 在当前目录生成很漂亮的结构树
::当前目录名获取==============
set “lj=%~p0”
set “lj=%lj:\= %”
for %%a in (%lj%) do set wjj=%%a

echo 文件夹是:%wjj%

tree /f > “%wjj%目录的结构树.TXT”
echo.

echo.

echo.
cls
echo　　　 完成，1秒后自动退出……
ping 127.0.0.1 -n 1 -w 1000 > nul

goto out

:out
cls
exit

:BB
CLS
rem 象DIR命令一样,不过每个下级目录文件也列出
::当前目录名获取==============
set “lj=%~p0”
set “lj=%lj:\= %”
for %%a in (%lj%) do set wjj=%%a

echo 文件夹是:%wjj%

tree /f > “%wjj%下级目录文件.TXT”
echo.

echo.

echo.
cls
echo　　　 完成，1秒后自动退出……
ping 127.0.0.1 -n 1 -w 1000 > nul

goto out

:out
cls
exit

:CC
CLS
rem 简洁地列出目录当前目录文件
::当前目录名获取==============
set “lj=%~p0”
set “lj=%lj:\= %”
for %%a in (%lj%) do set wjj=%%a

echo 文件夹是:%wjj%

tree /f > “%wjj%目录的目录树.TXT”

=================================
echo.
cls
echo　　　 完成，1秒后自动退出……
ping 127.0.0.1 -n 1 -w 1000 > nul

goto out

:out
cls
exit

=================================
:XX
CLS
rem 象DIR命令一样,不过每个下级目录文件也列出
echo 请拖动文件夹到此窗口……
echo 拖动文件夹后按确定继续……
set /p Tmp=
cd /d %Tmp%
tree /f >”%Tmp%目录的结构树.TXT”
echo.

=================================
echo.
cls
echo　　　 完成，1秒后自动退出……
ping 127.0.0.1 -n 1 -w 1000 > nul

goto out

:out
cls
exit
==============================

另外如果要生成下面这样的目录树结构应该怎么做呢？

|—书籍
|—图片
|—–mustang
|—–默认相册
|—文档
|—视频
|—音乐
|-Applications
|—Akamai
|—CrossOver
|—–ScottradeELITE

你需要先进入到需要列出的文件夹, 使用 cd 命令. 然后完整粘贴下面这条命令：

ls -R | grep “:$” | sed -e ‘s/:$//‘ -e ‘s/[^-][^\/]*\//–/g’ -e ‘s/^/ /‘ -e ‘s/-/|/‘

另外从一个博客看到下面的方法。作为借鉴：

在Mac OSX 系统默认是没有类似windows中的 tree命令，找到一条比较有意思的命令可以实现：

find . -print | sed -e 's;[^/]*/;|____;g;s;____|; |;g'

命令执行后的效果如下：

.

|__.classpath

|__.project

|____bin

| |____com

| | |____micmiu

| |____simple.idl

|____src

| |____com

| | |____micmiu

| |____simple.idl

为了方便使用，写一个alias 到~/.profile里:

alias tree="find . -print | sed -e 's;[^/]*/;|____;g;s;____|; |;g'"
